IupSetLanguage

Defines the language used by IUP.

Parameters/Return

void IupSetLanguage(char *lng); [in C]
iup.SetLanguage(language :string) [in Lua]

lng: Language to be used. Can have one of the following values:

  • "ENGLISH"
  • "PORTUGUESE"

default: "PORTUGUESE".

Affects

All elements that have predefined texts.

Example in C

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include "iup.h"

void main(void)
{
  IupOpen();
  IupSetLanguage("ENGLISH"); 
  IupMessage("IUP Language", IupGetLanguage());
  IupClose();
  return;
}
